# Reference Notes

이 폴더는 `data/references/` 원본 자료를 바로 이력서 문구에 덮어쓰지 않고, 재사용 가능한 md 노트로 분해해 정리한 공간입니다.

한 파일로 합치지 않은 이유:
- 자료 성격이 다릅니다. 연차별 self review, 최근 프로모션 문서, 현재 이력서 원문, 제출용 이력서 양식이 섞여 있습니다.
- 이력서 문구 작업에는 `연표`, `반복되는 핵심 성과`, `소스별 맥락`, `양식 제약`을 따로 보는 편이 유리합니다.
- 나중에 특정 항목만 업데이트하거나 근거를 다시 확인하기 쉽습니다.

구성:
- `source-index.md`: 어떤 원본을 읽었는지와 용도
- `career-chronology.md`: 연차별 흐름과 역할 확장
- `themes-and-metrics.md`: 반복되는 핵심 테마와 정량 성과
- `resume-template-notes.md`: 제출용 이력서 양식에서 읽은 제약
- `source-notes/*.md`: 원본별 요약 노트

추천 사용 순서:
1. `themes-and-metrics.md`
2. `career-chronology.md`
3. 필요한 경우 `source-notes/*.md`
4. 제출 포맷 확인이 필요할 때 `resume-template-notes.md`
